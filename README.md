# 🎙️ TTS_OFFLINE — Мгновенная озвучка текста без интернета

**Полностью оффлайн. Полностью под твоим контролем.**

Это не просто ещё одна обёртка над нейросетями. Это готовый голосовой инструмент, который
работает прямо на твоём компьютере, не отправляет твои тексты в облака и не требует
подписки. Просто скачай, запусти — и говори.

## 🧠 Что внутри

- **Потоковая озвучка в реальном времени** — речь начинается мгновенно, не нужно ждать генерации целого аудиофайла. Чанк
  за чанком, как живой разговор
- **Два мощных TTS-движка**: Piper (быстрый, легковесный) и Silero (качественный, с живыми интонациями)
- **REST API** — встраивай в свои проекты на Python, Go, JavaScript, C# — на любом языке
- **Интерактивный терминал** — меняй голоса на лету, ставь на паузу, повторяй фразы, вставляй многострочный текст. Всё в
  одном окне
- **Нормализация текста** — числа, даты, дроби, единицы измерения превращаются в слова автоматически. Для русского и
  английского
- **Свой словарь** — если модель произносит слово не так, как тебе нужно, просто добавь его в словарь (PC → "пэ-ка")
- **Готовый .exe** — для тех, кто не пишет код. Запустил и работает
- **Кроссплатформа** — Windows, Linux. Везде одинаково

## 🎯 Для кого

- **Разработчики** — встраивай TTS в свои проекты через REST API
- **Создатели голосовых ассистентов** — озвучка без облаков и задержек
- **Инди-разработчики игр** — озвучка персонажей бесплатно

### ⚙️ Системные требования

Для корректной работы приложения, должен быть установлен Microsoft Visual C++ Redistributable\2016_NEW_RUNTIME_C++, он
доступен
здесь [https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### ⚡Быстрый старт для разработчиков

1. Создать виртуальное окружение (обязательно иначе засорится глобальный Python).
2. Инициализировать проект: `python start.py` (создадутся каталоги для речевых моделей).
3. Скачать модели:
    - [piper](https://huggingface.co/rhasspy/piper-voices/tree/main), скачивать файлы `.onnx` , `.onnx.json` (например
      для работы модели `ru_RU-denis-medium` качать файлы `ru_RU-denis-medium.onnx` и `ru_RU-denis-medium.onnx.json`).
    - [силеро](https://models.silero.ai/models/tts/) скачивать файлы с расширением .pt (например модель `v5_5_ru`
      состоит из файла `v5_5_ru.pt`).

4. положить модели в каталог, модели piper `resources/models/piper`, модели silero в каталог `resources/models/silero`
5. Проверить что всё работает, запустить приложение в интерактивном режиме: `python cli.py run`.
   появится чат с моделью '>>>', напишите в нем текст и нажмите 2 раза `enter`, должно начаться
   воспроизведение. Если нужно выйти то введите `exit` в новой строке.
6. Подробная справка о работе с приложением `python cli.py`, покажет все доступные команды.
7. Сборка приложения в исполняемый файл `python builder.py` (гарантированно работает на текущей версии, если код не был
   модифицирован).

### 🔁Работа с сервером

1. Подъём сервера в терминале введите: `python cli.py run-server -p 8000` (проследите чтобы порт 8000 был свободен, или
   укажите другой, в этом примере 8000). Далее вся работа строится через http.
2. Посмотреть docs и дергать api ручки можно из: [http://localhost:8000/docs/](http://localhost:8000/docs/)
3. Посмотреть доступные модели и голоса [http://localhost:8000/tts/info/](http://localhost:8000/tts/info/)
4. Запуск приложения (речевого движка), в engine укажите доступную модель, например:
    - piper: [http://localhost:8000/start/?engine=piper](http://localhost:8000/start/?engine=piper)
    - silero: [http://localhost:8000/start/?engine=silero](http://localhost:8000/start/?engine=silero)
5. Произнести фразу: [http://localhost:8000/tts/say/?text=приветствую](http://localhost:8000/tts/say/?text=приветствую)
6. Остановка приложения (речевого движка): [http://localhost:8000/stop/](http://localhost:8000/stop/)
7. Остановка сервера [http://localhost:8000/shutdown/](http://localhost:8000/shutdown/)
